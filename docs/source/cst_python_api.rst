Tutorials
=========

With the goal of illustrating the utilization of the API, three example scripts
are provided along with the source code of the module (see "examples" folder):

* frequencySelectiveSurfaceExample.py
* patchAntennaExample.py
* waveguideFilterExample.py

Each of these examples makes use of different features of the API, and can be
used as a starting point to create a new file to work with a different problem.
